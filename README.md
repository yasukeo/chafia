# Café Chafia — Maquette de site

Maquette de démonstration pour **Café Chafia** (Témara), brunch & tea time, réalisée pour montrer ce qu'un vrai site pourrait apporter : visibilité Google, carte complète, et prise de contact en un clic.

⚠️ Ceci n'est **pas** le site officiel de l'établissement.

## Structure

```
.
├── index.html
├── css/
│   └── styles.css
├── img/
│   ├── logo-icon.png    (logo détouré, header)
│   ├── logo-full.png    (logo complet original)
│   └── interior.png     (vraie photo de l'intérieur)
├── js/
│   └── main.js
└── README.md
```

## Aperçu local

Ouvre `index.html` dans un navigateur, ou utilise un petit serveur local :

```bash
npx serve .
```

## Déploiement

Déployable tel quel sur GitHub Pages, Vercel ou Netlify — aucune étape de build.

## Contenu

- **Le concept** — sens du nom "Chafia", présence Instagram
- **La carte complète** — petits-déjeuners, brunchs & déjeuners, gourmandises, boissons (vrais tarifs)
- **L'ambiance** — vraie photo de l'intérieur (mur de brique, banquettes en velours)
- Avis clients, horaires et coordonnées (téléphone, Instagram, WhatsApp)

Le logo et la photo d'intérieur sont les vrais visuels de l'établissement. Les 4 photos d'illustration de la carte (petit-déj, brunch, gourmandises, boissons) sont générées (Higgsfield) pour la démonstration.
